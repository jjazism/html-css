Principais formatações

Negrito / destaque

nesta frase temos um termo em negrito usando a tag B (sem semântica)

nesta outra frase temos um termo em negrito usando a tag STRONG (semântica)

itálico / ênfase

Nesta frase usamos o termo em itálico usando a tag I (nao sêmantica)

nesta frase usamos o termo em italico usando a tag EM (sêmantica)

Texto marcado

Podemos tambem marcar textos com a tag MARK

Este texto está marcado

esse texto está marcado, mas com a cor violeta

Texto Grande / Pequeno

aqui temos um texto grande e aqui temos um texto pequeno

temos grandes usamos a TAG BIG, mas ela ja se tornou obsoleta, provavelmente pelo motivo de nao ter semantica (sem significado), ja a tag SMALL que é usada para deixar textos pequenos tem semantica e ainda é usada

Texto deletado / inserido

podemos marcar um texto deletado para dizer que ele deve ser lido, mas nao considerado // TAG DEL

podemos marcar um texto como inserido para dar uma enfase e dizer que ele foi adicionado depois // TAG INS

existe tambem o sublinhado // TAG U (nao semantica)

Texto Sobrescrito

para inserir coisas do tipo x20+3 usamos a TAG SUP para inserir coisas ACIMA

Texto Subscrito

para inserir coisas do tipo h20 usamos a TAG SUB para inserir coisas ABAIXO
